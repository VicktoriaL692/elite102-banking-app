C2C BANKING APP

This is a simple command‑line banking application that lets users create and manage basic bank accounts using a local SQLite database. It is designed to be easy to use and provides all the essential features of a small banking system.

MAIN FEATURES!

Create Accounts  
Users can open a new account by entering a name, password, and initial deposit. The account is saved permanently in the database.

Login System  
Users can log in using their name and password. Once logged in, they gain access to account actions.

Check Balance  
After logging in, users can instantly view their current account balance.

Deposit Money  
Users can add money to their account. The balance updates immediately and is saved to the database.

Withdraw Money  
Users can withdraw money as long as they have enough funds. The app prevents overdrafts and blocks withdrawals that exceed the balance.

List All Accounts  
From the main menu, users can view all accounts stored in the system, including ID, name, and balance. This is helpful for testing or administrative viewing.

Menu-Based Navigation  
The app uses simple menus to guide the user.  
Main menu: create account, log in, list accounts, exit  
Account menu: check balance, deposit, withdraw, logout

Automatic Database Handling  
The app automatically creates the database file and the accounts table if they do not exist. All changes are saved instantly.
